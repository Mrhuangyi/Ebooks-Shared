> 主要分享技术方面电子书，当然由于个人爱好，文学、哲学、小说、诗歌、散文也是我痴迷的。
题外话，不管读什么书，条件允许的话，建议都读纸质版，并且支持正版
以上内容有自己购买或他人转发，如有侵权，请立即联系本人，第一时间删除

# 为什么建立该仓库

这个仓库放了大部分自己已经读过，读了一部分或者说准备要读的电子书。并不局限于技术书。  
说起来，个人还是对人文类书籍兴趣比较大。这个仓库用于记录下自己的阅读历程和后续的阅读计划。    


我看电子书一般使用Mac的iBooks或者亚马逊的kindle。
